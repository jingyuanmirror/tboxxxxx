## 一、背景与目标

### 1.1 背景

知识库由三大内容板块构成：**上传的内容**、平台制作的内容、存下来的模版。本文档聚焦第一板块——「上传的内容」，即用户从本地或第三方来源导入的文件/素材的管理功能。

### 1.2 目标

- 让用户能够快速上传、查找、整理本地文件
- 提供灵活的视图切换与多维度排序/筛选
- 通过「专辑」机制实现轻量级层级管理
- 保持操作链路简洁，上传至查看一步可达

### 1.3 成功指标

| 指标 | 目标值 |
|------|--------|
| 单次上传完成率 | ≥ 95% |
| 文件检索耗时（已有文件） | ≤ 5s |
| 用户 7 日留存（上传功能）| ≥ 60% |

## 四、详细功能说明

### 4.1 视图模式

#### 4.1.1 列表视图（List View）

- 每行展示一条文件记录，信息密度高，适合快速浏览大量文件
- 列字段（可拖拽调整顺序）：

| 列名 | 说明 | 默认显示 |
|------|------|----------|
| 文件图标 + 名称 | 点击可预览/打开 | ✅ |
| 文件格式 | PDF / DOCX / MP4 … | ✅ |
| 所属专辑 | 支持多专辑标签 | ✅ |
| 上传时间 | yyyy-MM-dd HH:mm | ✅ |
| 文件大小 | 自动换算 KB/MB/GB | ✅ |
| 操作 | 更多菜单（⋯） | ✅ |

- 支持多选（Checkbox）+ 批量操作

#### 4.1.2 方块视图（Grid View）

- 每个文件以卡片形式展示，默认 4 列（响应式：2 列 / 3 列 / 4 列）
- 卡片内容：
  - 顶部：**缩略图**（图片/视频取首帧；文档显示格式图标 + 首页预览；音频显示波形占位图）
  - 底部：文件名（超长省略 + Tooltip）、文件格式 Badge、上传时间
- 悬停：显示操作快捷按钮（预览、更多操作）
- 支持多选（长按或点击左上角 Checkbox）

#### 4.1.3 视图切换

- 右上角切换按钮（列表图标 / 方块图标），状态持久化至用户偏好

---

### 4.2 排序与筛选

#### 4.2.1 排序

支持以下维度单字段排序，升/降序切换：

| 排序维度 | 说明 |
|----------|------|
| 上传时间 | 默认降序（最新在前） |
| 名称 | 按字母 / 拼音 A–Z |
| 文件格式 | 按格式名称字母序 |
| 文件大小 | 从大到小 / 从小到大 |

- 列表视图：点击列表头触发排序
- 方块视图：顶部「排序」下拉选择器

#### 4.2.2 筛选

支持多条件组合筛选，实时生效：

| 筛选维度 | 控件类型 | 可选值 |
|----------|----------|--------|
| 文件格式 | 多选 Checkbox | 图片 / 视频 / 音频 / 文档 / 压缩包 / 其他（动态来源于已有文件） |
| 所属专辑 | 树形多选 | 所有专辑（含未分类） |
| 上传时间 | 日期范围选择器 | 今天 / 本周 / 本月 / 自定义范围 |

- 筛选面板：点击工具栏「筛选」按钮展开；已激活筛选项在入口处以 Badge 数字提示
- 支持「一键清除全部筛选」

#### 4.2.3 搜索

- 顶部搜索框，支持按**文件名**关键词模糊搜索
- 搜索与筛选互不冲突，可叠加使用
- 无结果时显示空状态插画 + 「清除搜索条件」按钮

---

### 4.3 新建 / 上传功能

#### 4.3.1 入口

- 页面右上角主操作按钮：**「+ 新建」**（Primary Button）
- 点击后弹出下拉菜单：
  ```
  ┌──────────────────┐
  │  📁  新建专辑    │
  │  ⬆️  上传文件    │
  └──────────────────┘
  ```

#### 4.3.2 上传文件流程

**Step 1 — 触发上传**

- 点击「上传文件」或将文件拖拽至页面任意区域（Drop Zone 高亮提示）

**Step 2 — 文件选择**

- 弹出系统文件选择器，支持多文件同时选中
- 支持拖拽文件夹（自动展开为文件列表，文件夹名可选性地映射为专辑）

**Step 3 — 上传前配置（弹窗）**

```
┌─────────────────────────────────────────┐
│  上传文件                          [×]  │
├─────────────────────────────────────────┤
│  已选择 3 个文件                        │
│  ┌──────────┬────────┬────────────────┐ │
│  │ 文件名   │ 大小   │ 状态           │ │
│  ├──────────┼────────┼────────────────┤ │
│  │ 报告.pdf │ 2.3MB  │ 待上传         │ │
│  │ 图片.png │ 450KB  │ 待上传         │ │
│  │ 数据.xlsx│ 1.1MB  │ 待上传         │ │
│  └──────────┴────────┴────────────────┘ │
│                                         │
│  加入专辑（可选）  [选择专辑 ▼]         │
│                                         │
│           [取消]  [开始上传]            │
└─────────────────────────────────────────┘
```

**Step 4 — 上传中**

- 弹窗底部或右下角 Toast 显示整体进度条
- 每个文件单独显示上传进度百分比 + 剩余时间估算
- 支持取消单个文件或全部取消
- 上传失败的文件高亮显示错误原因 + 单独重试按钮

**Step 5 — 上传完成**

- 全部成功：Toast 提示「已成功上传 X 个文件」，列表自动刷新并滚动至新文件
- 部分失败：弹窗保留，高亮失败文件，提供「重试失败项」按钮

#### 4.3.3 新建专辑流程

1. 点击「新建专辑」→ 弹出轻量输入弹窗：专辑名称（必填）+ 描述（选填）
2. 支持选择父专辑（实现嵌套，最多 2 层）
3. 点击「创建」后专辑出现在左侧筛选面板与「所属专辑」选项中

---

### 4.4 文件操作

右键菜单 / 更多操作（⋯）包含：

| 操作 | 说明 |
|------|------|
| 预览 | 打开内置预览器（图片/PDF/视频/音频） |
| 重命名 | 行内编辑，按 Enter 确认 |
| 移动到专辑 | 弹出专辑树选择器 |
| 下载 | 触发浏览器下载 |
| 复制链接 | 生成带权限的分享链接（后续迭代） |
| 删除 | 二次确认弹窗，删除后可在「回收站」找回（保留 30 天） |

**批量操作**（多选后工具栏出现）：

- 批量移动到专辑
- 批量下载（打包为 ZIP）
- 批量删除

---

### 4.5 文件限制与规格

| 项目 | 规格 |
|------|------|
| 单文件最大尺寸 | 500 MB（可配置） |
| 单次最多上传文件数 | 50 个 |
| 支持格式 | 图片：JPG/PNG/GIF/WebP/SVG；文档：PDF/DOCX/XLSX/PPTX/TXT/MD；视频：MP4/MOV/AVI；音频：MP3/WAV/AAC；压缩包：ZIP/RAR/7Z |
| 不支持格式 | 可执行文件（.exe / .sh / .bat 等）|
| 文件名长度 | ≤ 255 字符 |

## 五、页面结构与交互流程

### 5.1 页面布局

```
┌─────────────────────────────────────────────────────────┐
│  知识库  >  上传的内容                    [+ 新建 ▼]    │
├──────────────┬──────────────────────────────────────────┤
│              │  🔍 搜索文件名...   [筛选 ①] [排序 ▼]   │
```

### 5.2 核心用户流程

```
用户进入「上传的内容」
       │
       ├── 浏览/查找文件
       │       ├── 切换视图（列表 / 方块）
       │       ├── 点击专辑树筛选分类
       │       ├── 使用筛选面板（格式/时间）
       │       └── 搜索文件名
       │
       ├── 上传文件
       │       ├── 点击「+ 新建」→「上传文件」
       │       ├── 拖拽文件到页面
       │       ├── 配置专辑归属（可选）
       │       └── 确认上传 → 进度反馈 → 完成
       │
       └── 管理文件
               ├── 单文件：右键 / 更多菜单
               └── 多文件：多选 → 批量操作工具栏
```

## 六、异常状态处理

| 场景 | 处理方式 |
|------|----------|
| 文件格式不支持 | 上传前校验，弹窗内标红提示，不进入上传队列 |
| 文件超过大小限制 | 同上，显示「文件超过 500MB 限制」 |
| 网络中断 | 上传失败标红，显示「重试」按钮，支持断点续传（后续迭代） |
| 重名文件 | 提示「已存在同名文件」，提供「覆盖 / 保留两者 / 取消」三选项 |
| 列表为空 | 空状态插画 + 「立即上传文件」CTA 按钮 |
| 无搜索结果 | 空状态 + 「清除筛选条件」链接 |

## 七、非功能性需求

| 类别 | 要求 |
|------|------|
| 性能 | 列表首屏（100 条）加载 ≤ 1s；文件列表支持虚拟滚动（万级数据） |
| 安全 | 文件上传前进行病毒扫描（集成云存储安全能力）；访问需登录鉴权 |
| 可用性 | 键盘可访问（Tab / Enter / Esc / Delete）；符合 WCAG 2.1 AA |
| 国际化 | 界面支持中/英文切换；文件名支持全字符集 |

## 八、后续迭代规划（Out of Scope for v1）

1. **分享与权限控制**：生成链接，设置有效期与密码
2. **版本管理**：同名文件自动保留历史版本

5. **第三方导入**：对接 Google Drive / Dropbox / Notion
6. **协作共享专辑**：多人共同维护同一专辑
